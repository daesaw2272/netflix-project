Streaming Platform Content Performance Analysis

Objective

Analyse Netflix catalogue data to understand content trends, genre distribution, and platform growth patterns.

Dataset

Netflix Movies and TV Shows Dataset (Kaggle)

Tools

Python
Pandas
Seaborn
Matplotlib

Key Insights

Movies dominate Netflix’s catalogue.
Content production surged after 2016.
Drama and comedy are the most common genres.
The United States produces the majority of titles.
